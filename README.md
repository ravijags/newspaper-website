# Newspaper Website

Tools: Python, JavaScript, HTML, CSS, MySQL, Bootstrap, Django

Full-stack Django web application for a newspaper platform.

• Implemented user authentication (login and registration)  
• Admin functionality for adding, editing, and managing news articles  
• Category-based article listing  
• Responsive frontend using HTML, CSS, JavaScript, and Bootstrap  
• Backend developed using Django framework and Django ORM  
• MySQL database used for data storage  
• Followed Django MVT architecture  
• Application deployed on cloud platform
